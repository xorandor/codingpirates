| Type                    | Direction        | Format (one line, ends with `\n`) | Fields                                                               | Example                              | Notes                                                                  |
| ----------------------- | ---------------- | --------------------------------- | -------------------------------------------------------------------- | ------------------------------------ | ---------------------------------------------------------------------- |
| `JOINED`                | Client → Server  | `JOINED;<playerName>`             | `playerName` = name of the player                                    | `JOINED;Allan`                       | Server broadcasts to everyone else that player joined                  |
| `PING`                  | Client → Server  | `PING`                            | *(none)*                                                             | `PING`                               | Server can respond with `PONG` to confirm connection is alive.         |
| `MESSAGE`               | Client → Server  | `MESSAGE;<text>`                  | `text` = the rest of the line after the first `;`                    | `MESSAGE;Hello!`                     | Treat everything after the first `;` as text (no escaping needed yet). |
| `BOXMOVE`               | Client → Server  | `BOXMOVE;<boxId>;<x>;<y>`         | `boxId` = integer, `x`/`y` = integers                                | `BOXMOVE;1;100;300`                  | Server should validate and then broadcast an update.                   |
